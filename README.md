# Resume Ranking System

This project is an AI-driven resume ranking system that ranks multiple resumes
against a job description using Natural Language Processing (NLP) techniques.

## Technologies Used
- Python
- Flask
- scikit-learn
- TF-IDF Vectorization
- Cosine Similarity
- HTML, CSS, JavaScript

## Features
- Upload multiple resumes at once
- Supports TXT, PDF, and DOCX formats
- Ranks resumes based on relevance to a job description
- Displays similarity scores for comparison
